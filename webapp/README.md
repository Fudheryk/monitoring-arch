# NeonMonitor Web

Interface web pour le système de monitoring NeonMonitor.

## Versionnement

La version suit le format sémantique : `{MAJOR}.{MINOR}.{PATCH}`

- **MAJOR** : Changements incompatibles
- **MINOR** : Nouvelles fonctionnalités rétro-compatibles  
- **PATCH** : Corrections de bugs

La version enrichie inclut :
- Hash git du commit
- Numéro de build CI
- Timestamp de build

Exemple : `1.0.0+a722137.42.2024-01-14T103000Z`
